# 重要：必须使用 scripts/install.sh 按正确顺序安装。
# 不要直接 `pip install -r requirements.txt`！
# 直接安装会让 pip 同时解析 torch / sglang / flashinfer，
# 解出来的版本经常互不兼容（pip 不感知 wheel 间的 CUDA 一致性）。
#
# 正确安装顺序（install.sh 已封装）：
#   1. PyTorch  (匹配本机 CUDA 的 wheel index)
#   2. FlashInfer (匹配 PyTorch 主次版本和 CUDA 的 wheel)
#   3. sglang[all] (从 PyPI)
#   4. 本文件中的 leaf 依赖（必须最后，否则可能反向覆盖 torch / sglang）
# ============================================================================
